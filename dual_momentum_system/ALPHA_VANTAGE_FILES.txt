================================================================================
INTEGRATION NOTES
================================================================================

The system uses a multi-source approach:
1. Primary: Yahoo Finance Direct (unlimited, free)
2. Backup: Alpha Vantage (your API key)

This means:
- Most downloads use Yahoo Finance (no API quota used)
- Alpha Vantage only used when Yahoo fails
- Automatic failover ensures high reliability
- Your 500 daily requests last much longer!
